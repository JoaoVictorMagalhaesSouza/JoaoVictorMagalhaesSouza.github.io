ANIN: Investment Analisys Environment

The purpose of this project is create an environment that can help the beginning investor to make more assertive decisions in their investments. The main idea is to create a Machine Learning model capable of predicting the closing price of a stock on the Brazilian Stock Exchange 1 day in advance. Under the guidance of Dr. Fernando de Souza Bastos, we create this project by following the following steps:

  • Constructing the Data Lifecycle: in this point, we refer specifically to the acquisition, treatment, exploratory analysis and feature engineering steps;
  • Building Model: here, our focus is to build an robust XGBoost model capable of making acceptable predictions of the closing price in advance;
  • Implementing Markowitz Theory: we also implement the Efficient Portfolio Theory that presents the investor with the highest percentage of return on an investment portfolio for a given level of risk;
  • Creating of User Interfaces: for the creation of the interfaces, we used the Shiny package of the R language for the production of the environments, although all part of Machine Learning is implemented in the Python language;
  • Disclosure of Environment: aiming at the free distribution and use of our environment for all interested parties, we created a containerized application with Docker and deployed it in a Google Cloud service.

Flask Application for Bitcoin Estimator

This project is the result of a work from the Machine Learning Engineering discipline, offered at the Federal University of Viçosa during graduation. The goal was to build a predictive application that used real-time data. The pipeline followed to create this application was:

  • Data Acquisition;
  • Data Preprocessing;
  • Feature Engineering;
  • Comparative study and implementation of models;
  • Creating the Flask application;
  • Containerizing the Flask Application and Deploying to Google Cloud;

The main objective that was to learn how to manipulate some cloud resources was satisfactorily met, although the metrics of the predictor models for Bitcoin are not so good.
